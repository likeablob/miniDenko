# minidenko-webui

## Project setup
```
apt install -y xxd
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
# this will copy gzipped assets to `../src/webui` in a C-header format.
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
